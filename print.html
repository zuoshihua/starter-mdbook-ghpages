mdBook GitHub Pages Starter

Introduction

This is a starter template for mdBook on GitHub Pages

Installation

Install Rust by following this page. Once done, install mdBook.

cargo install mdBook

Development

To start development server, run

mdbook serve

To build this book, run

mdbook build

This creates a book directory containing the rendered book.

Deployment

Add the following to .github/workflows/deploy.yml

name: GitHub Pages

on:
  push:
    branches:
      - main
  pull_request:

jobs:
  deploy:
    runs-on: ubuntu-22.04
    permissions:
      contents: write
    concurrency:
      group: ${{ github.workflow }}-${{ github.ref }}
    steps:
      - uses: actions/checkout@v4

      - name: Setup mdBook
        uses: peaceiris/actions-mdbook@v1
        with:
          mdbook-version: "0.4.8"
          # mdbook-version: 'latest'

      - run: mdbook build

      - name: Deploy
        uses: peaceiris/actions-gh-pages@v4
        if: github.ref == 'refs/heads/main'
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./book

Open the repo settings, click "Pages". Set the "Source" to "Deploy from a branch". Select gh-pages as the branch and / as the folder.
